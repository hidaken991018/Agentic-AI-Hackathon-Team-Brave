# Product Overview

**Life Compass** は、家計や人生設計を支援する AI 駆動型アプリケーションです。マルチエージェントシステムを活用して、ユーザーのファイナンシャルプランニングをインタラクティブにサポートします。

## Core Capabilities

- **AI エージェントベースのヒアリング**: Vertex AI Agent Engine と Gemini API を組み合わせた 2 段階パイプライン
  - FP Instructor Agent: ファイナンシャルプランナーとして質問・指示を生成
  - JSON Editor Agent: ヒアリング内容を構造化 JSON として管理
- **セッション管理**: ユーザーごとの会話履歴を維持し、継続的な対話を実現
- **Firebase 認証**: セキュアなユーザー管理とアクセス制御
- **スキーマ駆動バリデーション**: Zod を使った厳格な JSON データ検証

## Target Use Cases

- 家計状況のヒアリングと整理
- 人生設計のゴール設定とプランニング
- AI との対話を通じた財務アドバイスの提供
- ユーザー固有の財務データの構造化管理
- インタラクティブなクイズによる財務知識の学習

## Value Proposition

Google Cloud を活用した Agentic AI Hackathon プロジェクトとして、複数の AI エージェントを協調させて、従来の静的なフォーム入力では実現できない柔軟で自然な対話型ヒアリングを提供します。
